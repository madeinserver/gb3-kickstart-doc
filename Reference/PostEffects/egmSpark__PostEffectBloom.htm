Description

Bloom takes the bright pixels in this and allows their color to bleed out into the surrounding image, creating a sense of "glow".

The associated shared buffers are normally created by PostEffectsDefaultLibrary.

Parameters

float m_fThreshold

Pixels brighter than this value will participate in the bloom.

Expected range: [0,1]

float m_fStrength

Controls the overall effect of the bloom.

A value of 0 will have no effect.

Expected range: [0,20], but larger values are accepted, and negative values cause a darkening instead of brightening.

virtual bool Skip() const

Skip should return true if this effect pass can be skipped this frame.

This is a potential optimization; if an effect is enabled but its parameters are set in such a way that it will have no visible impact on the frame, Skip can be used to bypass its Render step automatically. The default implementation always returns false.
