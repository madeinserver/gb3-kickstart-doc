Inheritance graph for egmSpark::PostEffectDepthOfField

Description

This depth of field effect blurs the screen, except for surfaces that are at the desired focal depth, with a smooth transition to a blur in the foreground and/or background.

The associated shared buffers are normally created by PostEffectsDefaultLibrary.

Parameters

float m_fRange[4]

The range parameter controls four world-space distances.

0. The blurry foreground ends here, to fade to clarity by distance 1.

1. The completely clear range begins, continues to distance 2.

2. The clear range ends, fades to the blurry background by distance 3.

3. The blurry background begins here.

The four values should be in increasing order, or discontinuities will appear in the blur.

float m_fStrength

An overall strength of the blur.

0 produces no effect.

Expected range: [0,1]

float m_fNearStrength

Separately controls the foreground blur strength.

0 produces no blur.

Expected range: [0,1]

float m_fFarStrength

Separately controls the background blur strength.

0 produces no blur.

Expected range: [0,1]

bool m_bDiagnostic

Replaces the output shader with a diagnostic view of the ranges.

The foreground is rendered in red, and the background is rendered in green.

virtual bool Skip() const

Skip should return true if this effect pass can be skipped this frame.

This is a potential optimization; if an effect is enabled but its parameters are set in such a way that it will have no visible impact on the frame, Skip can be used to bypass its Render step automatically. The default implementation always returns false.
