Description

Color correction provides a few simple parameters for adjusting the color of the full screen.

The three parameters are applied in order:

1. Saturation

2. Multiply

3. Add

Parameters

float m_fSaturate

Fades the screen between greyscale and colorful.

A value of 1 has no effect (normal color).

A value of 0 produces a greyscale image.

Expected range: [0,2], but larger values are acceptable, and negative values have a complementary color effect.

NiColor m_kMultiply

Multiplies against the color of all pixels on the screen.

This can be used to adjust contrast, or to apply a color filter to the whole screen.

A valule of (1,1,1) has no effect.

A value of (0,0,0) produces a black screen.

A value of (1,0,0) produces a red filtered screen.

NiColor m_kAdd

Adds to the color of all pixels on the screen.

Expected range: [(0,0,0),(1,1,1)], but negative values are also acceptable.

virtual bool Skip() const

Skip should return true if this effect pass can be skipped this frame.

This is a potential optimization; if an effect is enabled but its parameters are set in such a way that it will have no visible impact on the frame, Skip can be used to bypass its Render step automatically. The default implementation always returns false.
