PostEffectsDefaultLibrary()

Creates a set of effects and adds them to the given PostEffectsStep.

It also creates the requisite shared buffers for these effects.

An instance of the PostEffectsDefaultLibrary may be deleted as soon as all desired information has been read from it (e.g. GetBloom()).
