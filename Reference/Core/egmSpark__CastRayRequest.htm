egmSpark::CastRayRequest

<PhysXHelperService.h>

Description

Contains the data necessary to make a ray cast request.

Public Attributes

efd::Point3 m_from

The origin point for the ray cast. This is required.

efd::Point3 m_direction

The direction vector of the ray cast. This is required.

efd::Float32 m_maxDist

The max distance for the ray cast. This is optional.

efd::UInt32 m_physXGroupMask

The PhysX collision group mask for collision groups that should be included in the ray cast.

This is optional.

egf::Entity* m_pIgnore1

An entity to ignore in the ray cast. This is optional.

egf::Entity* m_pIgnore2

Another entity to ignore in the ray cast. This is optional.

egf::Entity* m_pHitEntity

The entity that was hit by the ray cast.

NxActor* m_pHitActor

The PhysX actor which was hit by the raycast.

NxShape* m_pHitShape

efd::Float32 m_hitDistance
