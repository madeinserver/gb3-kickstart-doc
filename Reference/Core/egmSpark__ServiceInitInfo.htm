Inheritance graph for egmSpark::ServiceInitInfo

Description

This class defines information necessary to initialize a class.

It is assumed that you already know the ClassID, and this can be used in a map to hold the other info necessary to implement the class.

Public Attributes

efd::Bool m_initService

If true, we will initialize this service.

If false, don't init it at all. This is useful if you want to enable/disable certain services at runtime. For example, you may not want audio, or VisualTracker, or RapidIteration services on this particular run, but your code is built with them.

efd::UInt32 m_priority

Priority at which to initialize this service, if it is to be initialized.

If you specify the constant USE_DEFAULT_SERVICE_PRIORITY, then the default priority will be used.
