Description

ParticleEffect built-in model implementation.

This built-in model is used in conjunction with particle effects that are embedded within NIF files. The model implements useful functionality like the ability to destroy the entity when an effect is done. You may also specify an effect lifetime which will abort the effect after a period of time.
