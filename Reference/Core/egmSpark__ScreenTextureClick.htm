Description

ScreenTextureClick is a wrapper class for render clicks that are drawn in screen space, like HUDs and Splash Screens.

It uses the PreRender callback to update all elements before being rendered. It also adds the ability to find texture assets using the AssetLocator.

Internally it contains a single render view with multiple 2D mesh elements.
