Inheritance graph for egmSpark::PreloadGroup

Description

Helper struct.

This struct describes a set of URNs that were loaded together. The first item in the item list is the flat model URN that was given to the preloader. Subsequent items were asset URNs that were referred to by the flat model and therefore cascaded.
