Inheritance graph for egmSpark::GamebryoSplashState

Description

This state is responsible for displaying a gamebryo logo screen.

It is subclassed from the PreloaderState so you may optionally specify a PreloaderModel to preload models while the splash screen is displayed.

Pass to the constructor the GameState pointer of the game state to be called when this one is done
