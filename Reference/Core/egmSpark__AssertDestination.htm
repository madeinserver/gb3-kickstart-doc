void ResetIgnoredAsserts()

Reset the list of asserts to ignore.

If the user selects "ignore all" it will disable all further assert dialogs for future error messages with the same file name pointer and line number. .

If file name and line number tracking is not enabled then all asserts will be treated as coming from the same source and ignoring one will ignore all.
