Inheritance graph for egmSpark::BehaviorInvoker

Description

Template class for wrapping behaviors that associates the methods with the correct class.

This class is not used directly but rather is used by including a SPARK_BUILTINMODEL_BEHAVIOR macro inside the SPARK_BEGIN_DECLARE_BEHAVIORS block of a built-in model implementation.
