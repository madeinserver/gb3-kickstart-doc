Description

The Spark BuiltinUpdateService keeps track of the majority of builtin-models in the Spark libary.

Spark Built-in models register with the BuiltinUpdateService when they are created and unregister when they are removed.

The BuiltinUpdateService calls the update method on each Builtin model on every OnTick call of the service. It also registers for actor added and removed messages sent by the AnimationService. When actors are added or removed, the builtin models for that actor, are notified

void RegisterBuiltinModel(
const egf::EntityID& entityID,
IBuiltinModelBase* pBuiltinModel)

Registers a IBuiltinModelBase derived built-in model for the given owning entity with the with the BuiltinUpdateService.

Registration is required in order to have the various extended methods on the built-in model (such as Update, PostUpdate, or actor add/remove notifications) invoked. Multiple built-in models from the same entity can be registered.

Built-in models derived from IBuiltinModelBase will automatically call this method during IBuiltinModelBase::OnEndLifecycle at the completion of the OnCreate lifecycle.

Parameter Description
entityID the entityID of the built-in model's owning entity
pBuiltinModel the built-in model being registered

void UnregisterBuiltinModel(
const egf::EntityID& entityID,
IBuiltinModelBase* pBuiltinModel)

Unregisters the given built-in model instance on the specified owning entity from the BuiltinUpdateService.

Built-in models derived from IBuiltinModelBase will automatically call this method during IBuiltinModelBase::OnRemoved when the owning entity is removed from the EntityManager.

Parameter Description
entityID the entityID of the IBuiltinModelBase's owning entity
pBuiltinModel The built-in model to unregister

void PauseEntity(const egf::EntityID& entityID)

Pauses all registered built-in models for this entity.

These built-in models will not call Update() or PostUpdate() as long as the entity is paused.

void ResumeEntity(const egf::EntityID& entityID)

Resumes all registered built-in models for this entity.

These built-in models will start calling Update() and PostUpdate() again

void SetPaused(efd::Bool isPaused)

Pause or resume the entire BuiltinUpdateService.

When paused, no built-in models will have their update or post update functions called.
