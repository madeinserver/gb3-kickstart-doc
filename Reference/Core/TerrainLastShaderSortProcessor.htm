Description

This is a custom render list sorter based on NiShaderSortProcessor.

The PreRenderProcessList() method is overriden and additiona logic has been added to render terrain meshes after all other meshes (except those that have alpha blending).

Since terrain is often occluded by other objects, this saves GPU rendering time that would be spent drawing terrain pixels that would be eventually overwritten anyway.
