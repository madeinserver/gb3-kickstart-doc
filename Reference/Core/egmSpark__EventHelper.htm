egmSpark::EventHelper

<EventHelper.h>

Inheritance graph for egmSpark::EventHelper

Description

The main purpose of the EventHelper class is to enable the creation of a centralized place where events and their parameters can be declared.

This helper class enables Event parameters to automatically be streamed in/out of the DataStream when being sent& received.
