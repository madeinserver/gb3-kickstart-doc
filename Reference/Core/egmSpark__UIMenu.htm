Description

This class provides the interface to create menus using the UIManager and NiUserInterface.

Expected usage is to add each menu item using AddMenuItem, then call CreateUIElements.

template <class CallbackClass > void AddMenuItem(
const char* pMenuItemName,
CallbackClass* pInstance,
void(CallbackClass::*)(const efd::utf8string &) pfnCallback)

Handles the addition of a new menu item to the menu and sets up a callback method to handle the selection of that menu item.

The callback method is must be an instance method with the following signature:

void CallbackMethod(const efd::utf8string&);

Parameter Description
pMenuItemName The string that will be listed in the menu to represent this item
pInstance Pointer to the instance of class CallbackClass that will handle this item's selection
pfnCallback Pointer to the handler method for this menu item
