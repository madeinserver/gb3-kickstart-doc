Inheritance graph for egmSpark::SparkTerrain

Description

When mixed in with a terrain model, this builtin-model adds support for terrain fog (it adds the terrain's fog property to all scenegraphs at startup) In addition, the TerrainDepth material will be applied to the terrain in order to support depth of field (and other depth-related) effects.
