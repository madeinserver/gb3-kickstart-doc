Inheritance graph for egmSpark::CameraShake

Description

CameraShake model implements a simple camera shake for doing simple earth-quake effects Note: You will need to mix this into your SimpleCamera derived camera.

CameraShake()

void ShakeCamera(
const efd::Point3& shake,
efd::Float32 decayRate,
efd::Float32 duration,
efd::Bool accumulative)

Apply a shaking force to the camera.

Parameter Description
shake The maximum distance that the camera can shake along each axis.
decayRate The amount of decay in the magnitude of the shake force per second.
duration The amount of time the shaking will last in seconds.
accumulative Whether the new shaking force will be applied on top of any existing shaking force.

void OnShakeCamera(efd::DataStream* pStream)

Event handler function for shaking the camera.

Parameter Description
pStream A DataStream containing the parameters for the ShakeCamera function.

efd::Bool GetShake(
efd::Float32 fDelta,
efd::Point3& shake)

Gets the current shaking force to apply to the camera.

Parameter Description
fDelta The amount of time that has passed since the last call to this function.
shake Receives an offset to apply to the position of the camera in order to shake it.
Return Value True if the function succeeded.
