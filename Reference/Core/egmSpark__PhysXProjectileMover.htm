Description

Utilizes a Kinematic physX actor to move a projectile through a physX scene.

Sends a collision event when a collision occurs. User can create the physX actor via properties set in EMT or WB

Projectiles are not replicated. Instead, when a projectile is fired, an event is sent to the clients to create the projectile and simulate it locally.
