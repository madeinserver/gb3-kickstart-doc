egmSpark::AutoPerfTimer

<PerformanceTimer.h>

Description

A class to automatically start and stop a performance timer based on the scope of the instance.

This is intended to be used through the PERFORMANCE_AUTO_TIMER() macro.

Public Methods

AutoPerfTimer(PerformanceTimer& timer)

Constructor.

Starts the provided timer.

Parameter Description
timer The timer to start.

~AutoPerfTimer()

Destructor. Stops the timer that was started by the constructor.
