efd::Bool ComputeWalk(
egf::EntityID entity,
efd::Point3 goal,
efd::Point3& result)

Makes an entity conform to walkable and non-walkable areas.

This built-in enables an entity to conform to the height of walkables or terrain and be blocked and slide around non-walkables and terrain. It depends on the WalkableService to collect references to geometry that follows a naming convention marking it as walkable or non-walkable and optionally an NiTerrain object.

Parameter Description
entity ID of the entity that is trying to move
goal the desired new position of the entity
result return value indicating where the entity should end up after being blocked by non-walkables, sliding, and conforming to the height of the walkable surface
Return Value whether or not any valid potential results were found

void SetWalkAnywhere(efd::Bool walkAnywhere)

Toggles the walkability code on and off.

If WalkAnywhere is set to true, walkability will not interfere with characters' navigation. Useful for debugging.

Parameter Description
walkAnywhere Desired state of the WalkAnywhere flag.

efd::Bool GetWalkAnywhere()

Queries the current value of the WalkAnywhere flag (see SetWalkAnywhere).

Parameter Description
Return Value State of the WalkAnywhere flag.
