egf::EntityID SpawnEntityFromModel(
const efd::utf8string& modelName,
const efd::Point3& position,
const efd::Point3& rotation)

Spawn entity from model and specify the initial position and rotation.

The spawned entity will be tracked by the SpawnerService which will destroy the entity at shutdown or when SpawnerService::DestroyAllEntities is called.

Script Binding.

Parameter Description
modelName Name of the model.
position The position.
rotation The rotation.
Return Value The EntityID of the created entity.

egf::EntityID SpawnEntityFromModel(
const efd::utf8string& modelName,
efd::Bool trackEntity = true)

Spawn entity from model and specify whether the SpawnerService should track the entity.

If the entity is tracked then it will be destroyed when the SpawnerService is shutdown or when SpawnerService::DestroyAllEntities is called.

Script Binding.

Parameter Description
modelName Name of the model.
trackEntity If true, the SpawnerService will track this entity and will delete the entity when tracked entities are cleaned up.
Return Value The EntityID of the created entity.
