egf::EntityID CreateEntityIfDoesntExist(
efd::utf8string modelToCheck,
efd::utf8string modelToCreate,
efd::Bool issueWarning = false)

Creates an entity if no similar entity already exists.

Checks for any existing entity that mixes in the specified check model and if no match is found then it creates the specified create entity.

Script Binding

Parameter Description
modelToCheck The model to check for previously existing versions.
modelToCreate The model to create if no instance of the check entity exists.
issueWarning If true, an error will be logged if modelToCheck doesn't already exist.
Return Value The entity ID of the entity, either the existing entity that was found or the entity that was just created.

egf::EntityID EnsureActiveCamera(efd::utf8string modelToCreate)

Ensures that an active camera entity exists.

First it checks for an existing camera and if none is found it creates a camera entity of the given model type.

Script Binding.

Parameter Description
modelToCreate The model to create if there is not already an active camera.
Return Value The entity ID of the camera, either the existing camera that was found or the camera that was just created.
