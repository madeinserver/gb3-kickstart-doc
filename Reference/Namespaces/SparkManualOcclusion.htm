SparkManualOcclusion Namespace

Private Methods

efd::Bool HideRegion(efd::SInt32 region)

Makes all entities marked for a particular region non-visible.

Each entity that contains the property 'OcclusionRegion' with a value set to the value of the argument will have their 'Visible' property set to false. The intended use is for a trigger to be set between regions that will call this built-in to optimize the number of objects being drawn.

Script Binding.

Parameter Description
region identifier integer of the region to hide.
Return Value whether any entities were found for the given region.

efd::Bool ShowRegion(efd::SInt32 region)

Makes all entities marked for a particular region visible.

Each entity that contains the property 'OcclusionRegion' with a value set to the value of the argument will have their 'Visible' property set to true. The intended use is for a trigger to be set between regions that will call this built-in to optimize the number of objects being drawn.

Script Binding.

Parameter Description
region identifier integer of the region to make visible.
Return Value whether any entities were found for the given region.
