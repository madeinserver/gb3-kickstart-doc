Private Methods

egf::EntityID GetEntityByName(const efd::utf8string& name)

Gets the first entity with the given name.

To retrieve an entity by name, it must mixin the Nameable model.

Script Binding.

Parameter Description
name The name of the entiy.
Return Value The EntityID for the entity with that name. If no entity is found the invalid EntityID is returned.

efd::list<egf::EntityID> GetEntitiesByName(const efd::utf8string& name)

Gets a list of entity ids for every entity that has the given name.

To retrieve an entity by name, it must mixin the Nameable model.

Script Binding.

Parameter Description
name The name of the entiy.
Return Value A list of EntityIDs for every entity found with the provided name.

egf::EntityID GetClosestEntityInRange(
const efd::Point3& origin,
efd::Float32 radius,
const efd::utf8string* pModelName = NULL)

Finds the closest entity in range, given a starting origin and a radius from which to search.

By default, static entities are not considered in this search. To change this you will need to look at the LevelService. You can also optionally filter the search to only consider entities that mixin a specific model.

Script Binding.

Parameter Description
origin The origin.
radius The radius.
pModelName Name of the model that must be mixed into the entity.
Return Value The closest entity found in range. If no entity is found the invalid entity id is returned.

efd::list<egf::EntityID> GetEntitiesInRange(
const efd::Point3& origin,
efd::Float32 radius,
const efd::utf8string* pModelName = NULL)

Finds all the entities in a range, given a starting origin and a radius from which to search.

By default, static entities are not considered in this search. To change this you will need to look at the LevelService. You can also optionally filter the search to only consider entities that mixin a specific model.

Script Binding.

Parameter Description
origin The origin.
radius The radius.
pModelName Name of the model that must be mixed into the entity.
Return Value A list of entity ids for each entity found in a range.
