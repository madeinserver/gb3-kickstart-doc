Updating Kickstart from SVN

Emergent Labs distributes Spark and the Alpha Game sample through its Subversion (svn) server. Whenever Emergent pushes new code, levels, or bug fixes, they will go to the latest branch ( https://labs.emergent.net/src/latest).

To download Spark

  1. If you have not already done so, download and install TortoiseSVN.
  2. On your workstation, create a directory in which to download projects from Emergent Labs. For example, c:\Emergent\Labs.
  3. Within c:\Emergent\Labs, create a checkout directory for Spark. For example, c:\Emergent\Labs\Spark.
  4. In Windows Explorer, right-click the directory in which you want to download files. From the context menu, choose SVN Checkout.

    Figure 1

  5. In the Checkout dialog (Shown Below), enter the information as follows:
    • URL of repository = https://labs.emergent.net/src/latest/Spark
    • Checkout directory = c:\Emergent\Labs\Spark
    • Checkout Depth = Fully recursive
    • Revision = HEAD revision

    Figure 2

  6. Click OK.
  7. In the Authentication dialog, enter your Pulse credentials. These are the user name and password that you use to access Forums and Support on Emergent's community portal. Note that changing your password on Pulse also changes it for the svn server.

    Figure 3

To Download the AlphaGame Demo

  1. On your workstation, create a checkout directory for AlphaGame. For example, c:\Emergent\Labs\AlphaGame.
  2. In the Checkout dialog, enter the information as follows:
    • URL of repository = https://labs.emergent.net/src/latest/AlphaGame
    • Checkout directory = your directory here (EXAMPLE: c:\Emergent\Labs\AlphaGame)
    • Checkout Depth = Fully recursive
    • Revision = HEAD revision

To Download the DayDreamer Demo

  1. On your workstation, create a checkout directory for Multiplayer Game. For example, c:\Emergent\Labs\MultiplayerGame.
  2. In the Checkout dialog, enter the information as follows:
    • URL of repository = https://labs.emergent.net/src/latest/DayDreamer
    • Checkout directory = your directory here (EXAMPLE: c:\Emergent\Labs\DayDreamer)
    • Checkout Depth = Fully recursive
    • Revision = HEAD revision

It is not a requirement for your AlphaGame or DayDreamer checkout directory to be within the same parent directory as Spark.

If you choose not to run the installer, then remember that you will need to install any required software and set any required environment variables manually before you will be able to run AlphaGame and DayDreamer. Please see the Introduction section for a detailed list of required prerequisites
