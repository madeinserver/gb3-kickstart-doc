Introduction for Artists and Designers

KickStart gives artists and designers an easy and solid starting point for building a game based on Gamebryo LightSpeed. Using Kickstart's core functionality, artists and designers can build basic game functionality within just a few hours of work.

The tutorial for artists and designers walks you through the steps that you need to take to quickly create and run an application that is based on Kickstart.

First, we'll examine AlphaGame to help familiarize you with the Entity Modeling Tool (EMT) and the World Builder. We'll then move on to adding entity models and entity instances within a level. Throughout the tutorial, you'll be able to monitor the effects of your changes in a running application.

Tutorial at-a-Glance

These are the steps that you'll take in this tutorial:

  • Intro to Rapid Iteration - Modifying AlphaGame
  • Adding Entity Models
  • Adding Props to the Level
  • Creating a New Block file
  • Running a New Block file with AlphaGame
