Adding Props to the Level

World Builder is the level design tool for the Entity System and Game Framework in Gamebryo LightSpeed. In this section of the tutorial, you will use World Builder to populate your world with entities (sometimes referred to as "props" or "entity instances").

Adding entity instances (props) into the world

  1. If it is not already running, start Toolbench. From the Windows Start menu, select Emergent > Gamebryo LightSpeed > Tools > Toolbench.
  2. From the Toolbench menu, select File > Open Content and navigate to the directory that contains the Alpha Game solution file (for example, <AlphaGame>\Application\Data\AlphaGame.gsl).
  3. In the Game Solution View, double-click ?AlphaGame.block? (at the bottom of the tree).
    This opens the Alpha Game level within World Builder.

    Figure 1

  4. If you added any entity models since the last time you opened this .block file, you should be prompted to update your Palette View. You should choose to have the system update the Palette so that it contains all of your models, including the ones that you've added recently.
  5. From the Toolbench menu, select View > Palette View.
  6. From the Palette View, click and drag an entity into the world. When you release the mouse button, the entity is placed into the world.
    Tip: You can quickly create multiple instances of entities in the world by holding down the spacebar while you drag and click. In the following screenshot, you can see an entity called "MyCharacter0" placed in the world:

    Figure 2

  7. Optionally, you can override indvidual entity properties. For example, in the following screenshot, we've tweaked the Scale property that derives from the Placeable model. The "MyCharacter0" entity is now a more appropriate size for the world.

    Figure 3

  8. From the Toolbench menu, select File > Save All.
