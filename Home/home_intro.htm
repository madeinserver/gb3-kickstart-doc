Kickstart by Emergent Game Technologies

Kickstart demonstrates Gamebryo LightSpeed best practices, provides samples that are easy to follow and update, and includes useful library code in both C++ and scripting languages.

There are two parts to the Kickstart project:

  • Spark Library—provides basic services that are applicable to many different types of games. These services provide basic functionality and are designed to be extensible and modular.
  • Samples (sometime referred to as GamePacks)—genre-specific examples that use the Spark Library. These give you an easy way to quickly prototype any type of game by starting with the appropriate sample. Alpha Game is the first Kickstart sample that is generally available. This sample is a simple third person perspective game that exposes the basics of cameras, input handling, and character controls. It has been organized to make it easy to create a new game using original art, level design, and character behavior.

Features

The following features are included in Kickstart:

  • Easy install script - makes it easy to get a portable game up and running anywhere.
  • Organized and easy to understand structure
  • A Player Entity that is connected to a joypad or keyboard for input movements. Connections to the joypad, camera behavior, actions and animations can all easily be customized.
  • A Player Character setup an example character created in 3ds Max 9 as template for creating or importing new art.
  • NPC's that are driven by simple AI and follow waypoints.
  • A simple 2.5 D collision system for character height and obstacles.
  • Integrated sound via Wise entity project libraries.
  • Integrated Nvidia PhysX via a concise entity project library.
  • A state machine that drives games states like pause, win, loose, restart, etc.
  • A simple HUD system
  • An example trigger system that has opening doors, key items, etc.
  • Source art to all props and character files.
  • It's small and simple enough that it transfers fast and data can easily be swapped with more compelling data.

Prerequisites

This version of Kickstart requires the following other software to be installed:

  • Gamebryo Lightspeed 3.1.1.
  • PhysX 2.8.1 System Software (Optionally installed by Gamebryo installer).
  • Audiokinetic Wwise v2009.3 (Installed by the Kickstart installer).
  • LightSpeed Wwise Integration 2.3.1 (Installed by the Kickstart installer)

Additionally, DayDreamer requires:

  • Scaleform GFx 3.1.80 (Installed by the Kickstart installer).

To build from source code, you also need:

  • Microsoft DirectX SDK August 2009

Getting Updates

Emergent Labs distributes updates to KickStart through its Subversion (svn) server. Whenever Emergent releases new code, levels, or bug fixes, they will go to the latest branch. Please see the section on Updating Kickstart from SVN for more information.
